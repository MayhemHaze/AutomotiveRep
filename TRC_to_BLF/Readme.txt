É uma conversão de duas etapas, convertendo de .trc para .asc e posteriormente para .blf!

Ele abre uma janela gráfica, basta clicar em cima do botão e selecionar o arquivo desejado.

É possível haver alguns problema com o script, então garanta que essas 3 coisas estão funcionais:
- Python 3 deve estar instalado;
- Comando pip funcionando corretamente;
- Biblioteca "python-can" instalada.

"Para instalar a biblioteca python-can, use o seguinte comando no terminal: pip install python-can".

Abra o Código no VS Code e tenha as extensões de python devidamente instaladas.
